AYNA Cosmetics Website

This repository contains the official website for AYNA Cosmetics, a brand dedicated to minimal, modern, and intentional makeup. The website is built as a single-page application using pure HTML, Tailwind CSS, and vanilla JavaScript to simulate a multi-page experience.

Features

Multi-Page Feel in a Single File: Uses JavaScript to show and hide content sections, creating a fast, app-like navigation experience without page reloads.

Fully Responsive: Designed to look and work beautifully on all devices, from mobile phones to desktops.

Minimal & Elegant UI: The design follows the brand's aesthetic of "silent luxury" with a clean layout, soft nude color palette, and premium typography.

Brand Content:

Homepage: A stunning hero section with a clear call-to-action.

Shop: A product showcase with detailed descriptions.

Values: A clear breakdown of the brand's core principles.

Our Story: A vertical timeline detailing the brand's journey.

About Us: The brand's mission and "About Us" text.

Assets:

Includes the official logo.jpg.

Uses placeholder images for products, which can be easily replaced.

Project Structure

ayna-cosmetics.html: The complete and only file for the website. It contains all HTML structure, Tailwind CSS classes for styling, and JavaScript for interactivity.

logo.jpg: The brand's official logo image (you must add this file to the repository).

README.md: This file.

How to Run

There is no complex build process or server required.

Clone the repository:

git clone [https://github.com/your-username/ayna-cosmetics.git](https://github.com/your-username/ayna-cosmetics.git)


Add your assets:

Place your logo.jpg file in the root of the repository.

Replace the placeholder product image URLs (from placehold.co) in ayna-cosmetics.html with your actual product images.

Open the file:

Navigate to the cloned folder and simply double-click the ayna-cosmetics.html file to open it in your web browser.
